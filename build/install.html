HP Instant Ink · Usage Statement

Install the bookmarklet

Drag this button to your bookmarks bar:

📊 Instant Ink Usage

Can't drag? Create a new bookmark manually and paste the contents of build/bookmarklet.txt as the URL.

Then, to run it

  1. Log in to HP Smart and open HP Instant Ink → Print and Payment History (portal.hpsmart.com/.../print_plans/account_history).
  2. Click the Instant Ink Usage bookmark.
  3. It reads your billing cycles in your live session and shows an annual + monthly report. Use Copy report / Download JSON / Download CSV to export.

Runs entirely in your browser using your existing login. Nothing is sent anywhere except HP's own dashboard API, and no credentials are stored.
